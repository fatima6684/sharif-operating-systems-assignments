# HW8 - Operating Systems Practical Assignments

## Overview

This homework contains three practical exercises covering advanced operating system concepts:

1. Implementing a Reader-Writer Spinlock in the XV6 operating system.
2. Implementing a Docker-like container filesystem using OverlayFS and image layers.
3. Extending the custom filesystem to support concurrent access using synchronization mechanisms.

The goal of these exercises is to understand synchronization, container filesystem architecture, and challenges of concurrent filesystem design.

Each section contains a separate README file explaining the implementation details, design decisions, tests, and final results.

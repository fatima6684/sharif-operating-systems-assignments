# Homework 04 - Operating Systems

This homework focuses on thread context switching, Linux container namespaces, and user-space file system design.

## Topics Covered

- User-level thread implementation
- Context switching and CPU register management
- Cooperative thread scheduling
- Linux namespaces and process isolation
- PID, mount, UTS, time, and user namespaces
- Container environment customization
- User-space file systems with FUSE
- File system metadata and storage management

## Questions

### Q1 - User-Level Threads and Context Switching

Implemented a lightweight user-level threading library with context switching, thread scheduling, and cooperative multitasking.

### Q2 - Linux Container Namespaces

Explored Linux namespace mechanisms and extended a custom container runtime to support process, mount, UTS, user, and time namespace isolation.

### Q3 - User-Space File System using FUSE

Designed a simple user-space file system with custom metadata structures, file management operations, and persistent storage.

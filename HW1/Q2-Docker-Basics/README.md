# Docker Fundamentals and Container Exploration

## Description

This section introduces Docker and the fundamental concepts of container-based virtualization.

The experiments were performed using Alpine Linux containers.

---

## Experiments

The following tasks were completed:

- Installing Docker on Linux
- Pulling the Alpine Linux image
- Listing available images
- Creating and running containers
- Monitoring running containers
- Stopping and managing containers
- Executing commands inside a running container
- Comparing process information between host and container environments

---

## Investigated Concepts

- Docker Images
- Containers
- Docker Registry
- Container naming
- Container lifecycle
- Process visibility

Additional investigations include:
- Running Docker commands without `sudo`
- Listing stopped containers
- Using proxies for Docker image downloads

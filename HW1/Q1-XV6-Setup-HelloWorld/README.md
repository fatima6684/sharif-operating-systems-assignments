# XV6 Setup and Hello World Program

## Description

This part focuses on setting up the XV6 operating system and understanding the process of adding user-level programs.

XV6 was built and executed using QEMU, and the boot process of the operating system was inspected.

---

## Implementations

### XV6 Setup

The operating system environment was configured and XV6 was successfully compiled and executed.

The setup includes:
- Installing required dependencies
- Building XV6 source code
- Running XV6 using QEMU
- Verifying system boot

---

### User Program Development

A new user-space program was added to XV6 that prints:

```

Hello World!

```

The implementation required modifying:
- User source files
- Build configuration (Makefile)

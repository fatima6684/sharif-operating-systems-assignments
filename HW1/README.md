# Operating Systems Assignment 1

**Course:** Operating Systems  
**Instructor:** Dr. Hossein Asadi

## Overview

This assignment introduces fundamental operating system concepts through practical experiments involving XV6, Docker containers, and Linux kernel development.

The goal of this assignment is to gain hands-on experience with operating system internals, user-space and kernel-space programming, container technologies, and kernel module development.

---

## Contents

### 1. XV6 Setup and User Program Development

In this section, the XV6 educational operating system was configured and executed. A new user-level program was added to the operating system to print a simple message in the terminal.

Topics:
- Building and running XV6
- Understanding the XV6 development environment
- Adding a new user-space program
- Updating the build system

---

### 2. Docker Fundamentals

This section explores the basic concepts behind Docker and container technologies.

Topics:
- Docker installation and configuration
- Working with images and containers
- Executing commands inside containers
- Managing container lifecycle
- Comparing process information between host and container

---

### 3. Linux Kernel Compilation and Module Development

This section focuses on Linux kernel customization, enabling debugging features, compiling the kernel, and developing a simple loadable kernel module.

Topics:
- Downloading Linux kernel source code
- Configuring kernel debugging options
- Building a custom kernel
- Developing and testing a kernel module
- Inspecting kernel logs

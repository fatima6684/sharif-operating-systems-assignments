# Homework 08 - Operating Systems

This homework focuses on memory management in xv6, container runtime implementation, and file system security mechanisms.

## Topics Covered

- Dynamic memory allocation in xv6
- Implementation of `realloc()` and `calloc()`
- Memory testing and validation
- Docker container internals
- Container filesystem export and isolation
- Custom container runtime development
- User and group management
- Unix-style file permissions and access control

## Questions

### Q1 - Dynamic Memory Allocation in xv6

Extended the xv6 user memory allocator by implementing `realloc()` and `calloc()` functions and developed test cases to verify their correctness.

### Q2 - Docker Container Runtime

Investigated Docker container creation and filesystem behavior. Modified the container runtime to support executing containers from a custom base directory.

### Q3 - File System Access Control

Extended the file system with user, group, and permission management similar to Unix systems.
